# Homepage Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Build a focused first-pass homepage for Jeffrey Suffian using the official logo and core firm information.

**Architecture:** Keep the work inside `src/App.tsx` for this first homepage pass, reusing React and Tailwind utilities already present in the project. Store the official logo in `public/assets` so Vite serves it at `/assets/jeffrey-suffian-logo.png`.

**Tech Stack:** React 19, Vite, TypeScript, Tailwind CSS v4, lucide-react.
